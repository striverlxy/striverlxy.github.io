HSET

语法

HSET hash field value

功能

将哈希表hash中域field的值设置为value
如果哈希表hash不存在，将创建出新的hash并执行HSET操作
如果域field已存在，则新值会覆盖旧值

返回值

如果hash表中域field为新创建的，并成功设置value，则返回1
如果hash表中域field已存在，并成功用新值覆盖旧值，则返回0

代码演示

对不存在的hash表进行操作

127.0.0.1:6379> hexists student age
(integer) 0
127.0.0.1:6379> hset student age 10
(integer) 1
127.0.0.1:6379> hget student age
"10"

对已存在的hash表进行操作

127.0.0.1:6379> hexists student age
(integer) 1
127.0.0.1:6379> hget student age
"10"
127.0.0.1:6379> hset student age 20
(integer) 0
127.0.0.1:6379> hget student age
"20"

HSETNX

语法

HSETNX hash field value

功能

当且仅当field不存在于hash中时，才将field的值设置为value
如果field已存在于hash中，则赋值命令将被忽略
如果field不存在hash中，field的值被设置为value

返回值

执行成功时返回1
执行失败时返回0

代码演示

对field不存在时进行赋值

127.0.0.1:6379> hexists hash field 
(integer) 0
127.0.0.1:6379> hsetnx hash field 11
(integer) 1
127.0.0.1:6379> hget hash field
"11"

对已存在的field进行赋值

127.0.0.1:6379> hexists hash field
(integer) 1
127.0.0.1:6379> hget hash field
"11"
127.0.0.1:6379> hsetnx hash field 20
(integer) 0
127.0.0.1:6379> hget hash field
"11"

HGET

语法

HGET hash field

返回值

默认情况下返回给定域存储的值
如果域不存在，则返回nil

代码演示

域field不存在

127.0.0.1:6379> hexists hash2 field
(integer) 0
127.0.0.1:6379> hget hash2 field
(nil)

域field存在

127.0.0.1:6379> hset hash2 field 2
(integer) 1
127.0.0.1:6379> hget hash2 field
"2"

HEXISTS

语法

HEXISTS hash field

返回值

给定域存在的情况下返回1
不存在的情况下返回0

代码演示

127.0.0.1:6379> hexists hash3 field
(integer) 0
127.0.0.1:6379> hset hash3 field 1
(integer) 1
127.0.0.1:6379> hexists hash3 field
(integer) 1

HDEL

语法

HDEL hash field [field …]

功能

删除一个域或者多个域,如果域不存在，则被忽略

返回值

被成功移除的数量，被忽略的不包含在内

代码演示

127.0.0.1:6379> hset hash field1 11 field2 22 field3 33
(integer) 3
127.0.0.1:6379> hdel hash field2
(integer) 1
127.0.0.1:6379> hget hash field2
(nil)
127.0.0.1:6379> hdel hash field1 field3 field4
(integer) 2
127.0.0.1:6379> hget hash field1
(nil)
127.0.0.1:6379> hget hash field3
(nil)

HLEN

语法

HLEN hash

功能

返回hash的域的数量

返回值

返回域的数量
hash不存在时返回0

代码演示

127.0.0.1:6379> hset hash6 a1 11
(integer) 1
127.0.0.1:6379> hset hash6 a2 22
(integer) 1
127.0.0.1:6379> hlen hash6
(integer) 2

HSETLEN

语法

HSTRLEN hash field

功能

获取给定域field内存储的字符串value的长度

代码演示

127.0.0.1:6379> hset hash7 field aaaa
(integer) 1
127.0.0.1:6379> hget hash7 field
"aaaa"
127.0.0.1:6379> hstrlen hash7 field
(integer) 4

HINCRBY

语法

HINCRBY hash field increment

功能

为hash表中域field内保存的值加上增量increment, increment可以为负数
如果hash表不存在，则初始化出hash表再执行hincrby操作
如果field不存在，则先初始化出field，并将value初始化为0，再进行hincrby操作
如果field存储的为字符串，则操作报错
本操作的值被限制在 64 位(bit)有符号数字表示之内

返回值

进行增量后field内存储的值

代码演示

对不存在的field进行操作

127.0.0.1:6379> hexists counter page
(integer) 0
127.0.0.1:6379> hincrby counter page 10
(integer) 10
127.0.0.1:6379> hget counter page
"10"

对已存在的field进行操作

127.0.0.1:6379> hget counter page
"10"
127.0.0.1:6379> hincrby counter page 20
(integer) 30
127.0.0.1:6379> hget counter page
"30"

对存储为字符串的field进行操作

127.0.0.1:6379> hget counter page2
"asasas"
127.0.0.1:6379> hincrby counter page2 11
(error) ERR hash value is not an integer

HINCRBYFLOAT

语法

HINCRBYFLOAT hash field increment

功能

为hash表中域field内保存的值加上增量increment, increment可以为负数
如果hash表不存在，则初始化出hash表再执行hincrby操作
如果field不存在，则先初始化出field，并将value初始化为0，再进行hincrby操作
当域field存储的value不是字符串类型，会返回错误
当域field存储的value或增量increment不能被解释为双精度浮点数时报错

返回值

执行增量操作后域field内存储的value值

代码演示

对不存在的field执行操作

127.0.0.1:6379> hexists num float1
(integer) 0
127.0.0.1:6379> hincrbyfloat num float1 10.0
"10"
127.0.0.1:6379> hget num float1
"10"

两个普通的双精度操作

127.0.0.1:6379> hset num float2 10.0
(integer) 1
127.0.0.1:6379> hincrbyfloat num float2 12.0
"22"
127.0.0.1:6379> hget num float2
"22"

两个指数类型数据操作

127.0.0.1:6379> hset num float3 2e3
(integer) 1
127.0.0.1:6379> hincrbyfloat num float3 5e2
"2500"
127.0.0.1:6379> hget num float3
"2500"

HMSET

语法

HMSET hash field value [field value …]

功能

同时设置多个域field

返回值

设置成功，返回ok
如果设置不是hash表类型，报错

代码演示

127.0.0.1:6379> hmset website baidu baidu.com qq qq.com yahoo yahoo.com
OK
127.0.0.1:6379> hgetall website
1) "baidu"
2) "baidu.com"
3) "qq"
4) "qq.com"
5) "yahoo"
6) "yahoo.com"

HMGET

语法

HMGET hash field [field …]

功能

同时获取到多个域field存储的值

返回值

一个包含多个给定域的关联值的表，表值的排列顺序和给定域参数的请求顺序一样

代码演示

127.0.0.1:6379> hmset website baidu baidu.com qq qq.com yahoo yahoo.com
OK
127.0.0.1:6379> hmget website baidu qq yahoo xx
1) "baidu.com"
2) "qq.com"
3) "yahoo.com"
4) (nil)

HKEYS

语法

HKEYS hash

功能

返回hash表中所有的域

返回值

一个包含hash表中所有域的表
当hash表不存在时，则返回一个空表

代码演示

127.0.0.1:6379> hmset website baidu baidu.com qq qq.com yahoo yahoo.com
OK
127.0.0.1:6379> hkeys website
1) "baidu"
2) "qq"
3) "yahoo"
127.0.0.1:6379> exists website1
(integer) 0
127.0.0.1:6379> hkeys website1
(empty list or set)

HVALS

语法

HVALS hash

功能

返回hash表中所有域的值

返回值

一个包含hash表中所有值的表
当hash表不存在时，返回一个空表

代码演示

127.0.0.1:6379> hmset website baidu baidu.com qq qq.com yahoo yahoo.com
OK
127.0.0.1:6379> hvals website
1) "baidu.com"
2) "qq.com"
3) "yahoo.com"
127.0.0.1:6379> exists website1
(integer) 0
127.0.0.1:6379> hvals website1
(empty list or set)

HGETALL

语法

HGETALL hash

功能

返回hash表中所有的域和值

返回值

以列表形式返回hash表中的域和域的值
如果hash表不存在，则返回一个空的列表

代码演示

127.0.0.1:6379> hmset website baidu baidu.com qq qq.com yahoo yahoo.com
OK
127.0.0.1:6379> hgetall website
1) "baidu"
2) "baidu.com"
3) "qq"
4) "qq.com"
5) "yahoo"
6) "yahoo.com"
127.0.0.1:6379> exists website1
(integer) 0
127.0.0.1:6379> hgetall website1
(empty list or set)
